- Установить Java SDK 8 :
Скачать jdk 8...dmg для macOS с официального сайта Oracle
Запустить файл, дождаться завершения установки
Установить переменную окружения JAVA_HOME
	Открыть терминал
		touch .bash_profile
		open .bash_profile (откроется файл .bash_profile)

	Узнать путь к jdk:
		В терминале набрать /usr/libexec/java_home, нажать Enter - появится путь к jdk
	В .bash_profile ввести export JAVA_HOME=[путь, полученный на предыдущем шаге], сохранить

Проверить, что переменная окружения установлена правильно:
	В терминале набрать 
		source .bash_profile (загрузить файл)
		echo $JAVA_HOME
		Проверить, что путь к jdk правильный

- Создать Capabilities для iOS:
	Запустить Appium
	Стартовать сервер - откроется окно создания Capabilities
	Создать набор свойств
		platformName  iOS
		platformVersion 11.3 (должно соответствовать версии симулятора в XCode)
		deviceName iPhone 11.3 (должно соответствовать названию симулятора в XCode)
		app полный путь к приложению Wikipedia.app
		Сохранить Capabilitie с подходящим названием, например, Wikipedia iPhone 11.3
Запустить симулятор - он запускается, устанавливается приложение Wikipedia и WebDriverAgent
